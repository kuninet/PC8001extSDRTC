# hardware — 回路図・基板(設計メモ)

PC-8001 外部CPUバスに直結する PIC18F47Q43 ボードのハードウェア設計です。
回路図・基板は KiCAD で起こす予定です(本ディレクトリに追加)。

- **[design.md](design.md) … 回路設計仕様・結線表(フェーズA)。基本方針・ブロック図・
  ピンアサイン表・ネット単位の結線表・BOM。KiCAD はこの結線表をもとに引く。**

## 接続するバス信号(PC-8001 拡張バス → PIC)

- アドレス: `A0`-`A7`(I/O ポートデコード用。`D0H`-`D6H` を識別)
- データ: `D0`-`D7`(双方向。PIC が `IN` 時にドライブ)
- 制御: `/IORQ`, `/RD`, `/WR`, `/RESET`
- `/WAIT`: **PIC が駆動**(オープンコレクタ/オープンドレインで Wired-OR)

## PIC 側ペリフェラル

- microSD: ハードSPI(MSSP1)。CS/SCK/MOSI/MISO。
- (将来)RTC: I2C(MSSP2)。DS3231 等。SDA/SCL に 4.7kΩ プルアップ。
- I/O デコード: CLC(Configurable Logic Cell)で `A7..A0 == D0H..D6H` かつ `/IORQ` を
  ハード判定し、応答レイテンシを詰める。

## 検討事項(要実機検証)

- **/WAIT を PC-8001 拡張バスが尊重するか**(最優先)。尊重しない場合の回避策はソフト側で。
- データバス バッファ(74xx245 等)の向き制御を `/RD`・デコード出力で行うか、PIC直結か。
- 5V(PC-8001)と 3.3V(SD/PIC)のレベル変換。PIC18F47Q43 は 5V 動作可だが、SD は 3.3V。
- 拡張バスのバスタイミング(セットアップ/ホールド)の実測(ロジックアナライザ)。
